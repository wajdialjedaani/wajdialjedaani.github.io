CSCE 5270 - Computer-Human Interfaces

May 1, 2021

Course Description

Emphasizes human performance in using computer and information systems. Topics for software psychology include programming languages, operating systems control languages, database query facilities, computer-assisted dialogues, personal computing systems, editors, word processing and terminal usage by non-skilled users. Prerequisite(s): None.
